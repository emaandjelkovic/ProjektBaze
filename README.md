# AccountManager 

Projekt je napravljen u sklopu kolegija Teorija baza podataka.

Za potrebe projekta potrebno je u appsettings.json upisati svoj username i password te port kako bi projekt mogao raditi lokalno i povezati se na bazu.

Baza se može kreirati kroz visual studio u package manager konzoli naredbom "update-database" ili pokretanjem skripte u nekom od PostgreSQL management programa poput pgAdmin4, koji je korišten tijekom izrade ovog projekta.

Skripta za bazu se nalazi u root folder u projekta, te je i priložena u sklopu projekta u zasebnom folderu pod nazivom baza.sql.
